# Ping Pong Mini

Juego básico de ping pong (1 jugador vs CPU) en HTML + Canvas.

## Qué es ahora
Este repo es el **hub** de mini‑apps y juegos. La portada lista cada app y enlaza a sus páginas internas.

## Demo (GitHub Pages)
https://wincrash91.github.io/ping-pong-mini/

## Ejecutar
Abre `index.html` en tu navegador (o entra en la URL de Pages).

## Controles
- **W/S**: mover la pala
- **Espacio**: sacar

## Apps incluidas
- Pingpong (juego)
- PanelKit Calc (calculadora de materiales)
